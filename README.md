# ✏️ shopifyStore

## 📌 기본 소개

- Next에서 제공하는 [커머스 데모](https://github.com/vercel/commerce)를 모티브로 한 프로젝트
- webpack과 babel을 활용한 기본적인 프론트엔드 개발 환경 구축
- Redux를 사용한 상태 관리
- Jest, React-Testing-Library를 사용한 TDD 방식으로 개발 진행 중

### 📍 사용 기술

📎 **Front-end**

- React.js(v18)
- react-router-dom(v6)
- Redux Toolkit
- Jest / React Testing Library
- Emotion

📎 **Back-end**

- Firebase(REST)
